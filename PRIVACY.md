# Privacy Policy for DOM2MD

**Last updated:** May 2026

## Overview

DOM2MD is a Chrome DevTools extension that converts selected DOM elements to Markdown. It is designed with privacy in mind.

## Data Collection

**DOM2MD does not collect, store, or transmit any user data.**

Specifically:
- ❌ No personal information is collected
- ❌ No browsing history is tracked
- ❌ No data is sent to external servers
- ❌ No cookies or local storage are used for tracking
- ❌ No analytics or telemetry

## How It Works

The extension operates entirely locally within your browser's DevTools. It reads the HTML of the currently selected DOM element, converts it to Markdown using the [Turndown](https://github.com/mixmark-io/turndown) library, and displays the result in a sidebar panel. All processing happens on your device.

## Permissions

The extension requires the following Chrome permission:
- `clipboardWrite` — Used solely to copy the generated Markdown to your clipboard when you click the "Copy" button.

## Changes

If this privacy policy changes, updates will be posted in this file and in the Chrome Web Store listing.
