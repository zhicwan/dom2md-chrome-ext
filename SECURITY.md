# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in DOM2MD, please report it responsibly.

**Do NOT open a public GitHub issue for security vulnerabilities.**

Instead, please email: **imwosz@hotmail.com**

Include:
- A description of the vulnerability
- Steps to reproduce
- Potential impact

I will respond within 48 hours and work with you to address the issue.

## Scope

This policy applies to the DOM2MD Chrome extension source code in this repository.
